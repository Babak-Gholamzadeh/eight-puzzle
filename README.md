# Eight Puzzle
Solving 8 Puzzle problem by using Breadth-first search (BFS) algorithm (with some optimization) that can find the shortest solution in the fastest way.

The puzzle actually has two complete different answers and the domains of them are completly seperated and have no overlap with each other, So the program find the correct answer based on the inputed state.

### First answer:
0 1 2  
3 4 5  
6 7 8  

### Second asnwer:
1 2 3  
8 0 4  
7 6 5
